    สิ่งที่ได้รับจากเทอมที่ผ่านมา คือ ผมได้เรียนรู้เกี่ยวกับการทำเว็บหลายอย่างมากๆ เริ่มตั้งแต่การเปลี่ยนสี เปลี่ยนรูปร่าง การจัดตำแหน่งต่างๆ ผมรู้สึกว่าวิชานี้มีการสอนที่แปลกใหม่ดี ไม่ว่าจะเป็นการ pvp, ระบบการ์ด และระบบ guild ที่ทุกคนได้มาทำงานร่วมกันหรือมาพูดคุยปรึกษากัน ในส่วนของโปรเจค ผมได้ทำหน้าที่ในการเขียน code ของ landing page ซึ่งผมก็ได้ประยุกต์ใช้เนื้อหาที่ได้เรียนมา รวมไปถึงการศึกษาเพิ่มเติมเองอีกด้วย เพื่อนๆในกลุ่มก็คอยช่วยเหลือกันดีมากๆ ผมรู้สึกภูมิใจมากๆที่ได้ร่วมงานกับทุกๆคน และก็ภูมิใจกับงานที่เราได้ร่วมกันทำมามากๆ มันทำให้ผมรู้จักกับการทำงานเป็นทีมมากขึ้น อีกทั้งยังทำให้ผมจัดสรรเวลาได้เป็นระบบมากขึ้น โดยในเว็บไซต์นี้ที่ผมจัดทำขึ้นมามีการใช้เทคนิค hover, css var, css calc(), hsl และ transform ครับ อาจจะไม่ได้ใช้เทคนิคที่เรียนมาครบทุกอย่าง แต่ว่าผมก็พยายามเอาสิ่งที่ผมจำได้ และสามารถทำได้มาใส่ไว้ในเว็บนี้แล้วครับ
